# 🎮 Python Small Games

## 📌 Author

Oskar Jasłowski

This repository contains a collection of small games created in Python as independent mini-projects.

---

## 🎯 Project Overview

**Python Small Games** is a set of simple 2D games built using Python.

Each game focuses on:
- game development fundamentals  
- working with graphics, sound, and assets  
- real-time user input  
- practicing project structure and code organization  

---

## 🧩 Included Games

### 🏃 Ball Runner

Endless runner-style game where the player avoids obstacles and tries to survive as long as possible.

Includes:
- animations  
- sound effects  
- custom graphics and fonts  

---

### 🐍 Snake

Classic Snake game with:
- custom sprites  
- food system  
- sound effects  
- directional movement  

---

### 🏓 Pong

Simple implementation of the classic Pong game.

Features:
- two paddles  
- ball physics  
- basic collision system  

---

### 🐹 Kręci Drill

Arcade-style game with:
- custom sprites and environment  
- multiple game elements (pipes, ground, background)  
- structured code (`settings.py`, `sprites.py`)  

---

## 🖥️ How to Run

1. Navigate to a selected game folder:

```bash
cd game-folder-name
```

2. Run the game:

```bash
python code.py
```

## 📦 Technologies

- Python 3  
- Pygame (for graphics, input, and sound)  
- Custom assets (images, fonts, audio)
